Como os valores estão sendo colocados de forma aleatória, assim que recarrega a página ou clica no botão para resetar é feito uma jogada.

Isso acontece porque as jogadas nessa página são feitas de forma aleatória, chamando o método jogar sempre que a página é recarregada.

No arquivo exemplo.js pode fazer a modificação da variavel jogoEstatico para true para passar os valores dos dados manualmente, assim fica melhor pra testar o script.
